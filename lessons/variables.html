In this lesson you will learn how to use variables.

Variables are used to store data in them.
There are 3 types of variables. But we will only work with "let" for now.
Here is a example on how to define a variable:
let myVariable = 1;

The variable is called "myVariable" and contains the number "1".
Variables can also hold more than just numbers. Here for example it contains the text "Hello World!":
let myVariable = "Hello World!";
